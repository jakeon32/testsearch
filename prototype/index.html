PLAN셔틀버스 서치폼 (듀얼 모드)

화이트라벨 관리자 설정 기반, 노선/날짜 중심 전환 가능

핵심 콘셉트

관리자가 서비스 특성에 맞는 서치폼 모드를 선택하면, 화이트라벨 프론트엔드에 해당 모드가 적용됩니다. 동일한 데이터 소스를 사용하며, 검색 결과 페이지는 모드 무관하게 동일합니다.

화면 구성 요소

  • 1

    관리자 모드 토글

    화이트라벨 관리자 패널에서 "노선 중심" 또는 "날짜 중심" 모드를 선택합니다. 실제 서비스에서는 관리자 설정으로 반영됩니다.

  • 2

    편도/왕복 탭

    편도 또는 왕복을 선택합니다. 왕복 선택 시 귀환 날짜 필드가 추가됩니다. 기본값은 관리자 설정에 따릅니다.

  • 3

    가로형 서치바 (PC)

    필드 클릭 시 해당 필드 아래에 드롭다운 패널이 앵커링됩니다. 모바일에서는 바텀시트 모달로 전환됩니다.

  • 4

    날짜 선택 (캘린더 드롭다운)

    Mode A: 출발지/도착지 선택 후, 해당 노선 운행일만 활성화.
    Mode B: 전체 노선 운행일 합집합으로 활성화. 비운행일은 회색 처리.

  • 5

    출발지/도착지 선택

    출발지: 반드시 수동 선택 (자동선택 불가).
    도착지: 1개인 경우 자동 입력 + "자동" 뱃지 표시. 2개 이상이면 수동 선택 필수.

  • 6

    인원 선택 / 검색 실행

    기본 1명, +/- 버튼 조정. 필수 항목 모두 선택 완료 시 검색 버튼 활성화. 검색 결과 페이지로 동일한 파라미터를 전달합니다.
